# Customer Journey Map

**Project Name:** Vocal and Behavioral Candidate Evaluation System (VBCUA)
**Team ID:** Solo Developer
**Date:** 15 March 2024

## Journey Phases

| Phase | User Actions | Touchpoints | Emotions | Pain Points | Solutions |
|---|---|---|---|---|---|
| **Awareness** | Discovers VBCUA through academic channels or GitHub. | Social Media, GitHub | Curious | Difficulty finding tools that evaluate speech. | Clear README and documentation. |
| **Consideration** | Reviews the semantic and acoustic capabilities of the dashboard. | Streamlit App Landing Page | Hopeful | Unsure if local execution is easy. | Easy setup instructions and intuitive UI. |
| **Purchase/Use** | Uses the open-source Streamlit app locally, recording audio. | Microphone, Streamlit UI | Excited | Audio processing can be slow. | Use of Whisper `tiny` for fast local transcription. |
| **Retention** | Practices multiple times, tracking filler words and semantic scores. | Dashboard Analytics, PDF Reports | Confident | Forgetting past performances. | PDF generation allows them to save records. |
| **Advocacy** | Recommends the tool to peers for interview prep. | Word of Mouth, GitHub Stars | Satisfied | N/A | N/A |
