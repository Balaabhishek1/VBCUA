# Functional Features

**Project Name:** Vocal and Behavioral Candidate Evaluation System (VBCUA)
**Team ID:** Solo Developer
**Date:** 01 July 2026

## Implemented Features

| Feature | Description |
|---|---|
| **Audio Capture** | Users can record directly from the browser or upload `.wav`/`.mp3` files. |
| **Local Transcription** | Accurate speech-to-text powered by OpenAI Whisper. |
| **Semantic Scoring** | Cosine similarity scoring comparing candidate answers to academic references. |
| **Acoustic Analytics** | Calculation of Root Mean Square (RMS) energy and Zero-Crossing Rate (ZCR). |
| **Filler Word Detection** | Identification of hesitations (um, ah, like) using NLP. |
| **AI Feedback** | Qualitative coaching generated by Gemini Pro. |
| **PDF Reporting** | Downloadable comprehensive PDF reports containing text, metrics, and waveform plots. |
| **Database History** | Persistent storage of past evaluations using SQLite. |
