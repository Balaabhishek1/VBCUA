# Demonstration of Proposed Features

**Project Name:** Vocal and Behavioral Candidate Evaluation System (VBCUA)
**Team ID:** Solo Developer

## Demo Flow
1. **Launch:** Start the Streamlit application.
2. **Dashboard Overview:** Present the glassmorphic UI and explain the dual-focus (semantic + acoustic).
3. **Execution:** Record a sample interview response directly in the browser.
4. **Analysis:** Show the system processing the audio in real-time.
5. **Results:** Walk through the transcribed text, similarity score, filler words, acoustic waveform, and Gemini feedback.
6. **Export:** Download the PDF report to showcase the final deliverable.
